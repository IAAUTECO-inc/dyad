# Support & Reporting for DYAD

## Technical Support
If you encounter technical issues, please:
1. Search through existing [Issues](https://github.com/your-repo/dyad/issues).
2. Use the **Bug Report** template to provide structured data.

## Content & Ethics Reporting
As an AI-driven project, we take the output and use of our agents seriously. 
If you encounter:
- Offensive or harmful AI-generated content.
- Misuse of the framework that violates our Code of Conduct.
- Ethical concerns regarding the project's direction.

**Please report this directly to the Repository Administrators** via the GitHub "Report Content" feature or by emailing [admin-email@your-domain.org]. We guarantee a confidential review of all reports.

## Security Vulnerabilities
Please refer to our [SECURITY.md](./SECURITY.md) for instructions on reporting sensitive vulnerabilities. **Do not use public issues for security flaws.**
